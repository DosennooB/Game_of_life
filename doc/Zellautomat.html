    Zellautomat (game_of_life v0.3.5) View Source

    Logic des Zellautomaten.

    Die nötigen Paarameter werden in Agents gespeichert. Der Zelleautomat wird als Genserver Implementiert und von einem Supervisor überwacht. Er kann abstürtzen ohne das ein Datenverlust statt findet.

    tick() :: :ok

    Lässt den nächsten Zustand des Zellautomaten berechnen.

    Lässten nächten Zustand berechnen und gibt die neuen Werte an die Agents weiter. Nur Zellen die den Wert 1 haben oder Nachtbar einer Zelle mit dem Wert 1 sind, werden berechnet.
